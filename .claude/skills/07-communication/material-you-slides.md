# Material You Slides

Use this guide when the user wants a polished slide deck with a clean Material You visual
language.

## Best For

- architecture overviews
- experiment reviews
- training or serving system walkthroughs
- project updates for AI infra teams

## Design Defaults

- self-contained HTML slides
- 1280x720 layout
- Material You color-token structure
- Roboto typography
- rounded cards and surface hierarchy
- generous spacing instead of dense slide packing

## Slide Set Structure

Use a simple deck arc:

1. title slide
1. section divider when the story changes
1. content slides with one main claim each
1. closing or next-steps slide

## Content Rules

- keep one major message per slide
- use metrics, small tables, or diagrams instead of long paragraphs
- if showing AI infra performance, include the workload and hardware context
- if showing architecture, distinguish control plane from execution plane explicitly

## Review Rules

- the deck should help someone make a decision, not just admire formatting
- design polish should not hide assumptions, workloads, or baselines
