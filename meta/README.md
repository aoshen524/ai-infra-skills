# Meta Docs

This directory holds repo-maintenance or historical process docs.

These files are intentionally kept out of the main runtime path for Claude Code and
Codex agents.

Use them only when maintaining the repo itself:

- `HANDOFF.md`
- `REMAINING_TASKS_PROMPT.md`

Normal task execution should start from:

- `INDEX.md`
- `EXPERT_KNOWLEDGE_MAP.md`
- `.claude/skills/*`
- `knowledge/*`
